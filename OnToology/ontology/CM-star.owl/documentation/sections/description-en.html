Conceptual Mapping - RDF-star: Description back to ToC

Figure shows how this vocabulary extends the Conceptual Mapping to allow the generation of RDF-star graphs. Statements can be quoted using the properties `hasQuotedSubject` and `hasQuotedObject` for the positions of subject and object respectively. The quoted statement maps can be either `AssertedStatementMap` or `NonAssertedStatementMap`. Both `AssertedStatementMap` and `NonAssertedStatementMap` and in turn quote more Statement Maps, as they are subclasses of `cm:StatementMap`.
